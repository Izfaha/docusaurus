## Enable Tun

How to enable tun module in arch linux?
To verify you are able to hit these commands below.

```sh 
lsmod | grep tun
ls -l /dev/net/tun
```

it will show you tun if show nothing, it means your module is disable.
To check whether tun module is available in your system, hit this command.

```sh
modinfo tun
```
